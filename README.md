# bearbone

`bearbone` is a strongly typed, dynamic scripting language implemented in Scala.

<div style="float: left; margin-right: 45px">
  <img src="logo/bearbone-lang-logo-small.png" alt="bearbone-lang logo" style="width: 140px; display: block; margin: 0px;">
</div>

* Simpler than Python. 

* Safer than Javascript.

* Embeddable in Scala at runtime.

* Excellent for in-game scripting in [bearlyb](https://github.com/lego-eden/bearlyb).
