# bearbone

<img src="logo/bearbone-lang-logo-small.png" alt="bearbone-lang logo" style="width: 140px; margin: 0px">

`bearbone` is a bare-bone, strongly typed, dynamic scripting language implemented in Scala.
* Simpler than Python. 
* Safer than Javascript.
* Embeddable in Scala at runtime.
* Excellent for in-game scripting in [bearlyb](https://github.com/lego-eden/bearlyb).
